# Pokedex Language

포켓몬 도감 - 다국어 버전

## 주요 기능 (Features)

- **포켓몬 데이터 조회**: [PokeAPI](https://pokeapi.co/)를 연동하여 최신 포켓몬 데이터를 실시간으로 조회합니다.
- **포켓몬 타입 상성계산기** : 포켓몬 게임 내 배틀에서 정해진 규칙을 복잡하게 계산하거나 이미지를 따로 찾아보지 않고도 클릭 몇 번만으로 손쉽게 파악이 가능합니다.

## 추가 예정 기능 (Later)
- **다국어 지원**: 다양한 언어로 포켓몬의 이름과 정보를 확인할 수 있습니다. (추가 예정)
- **포켓몬 기술 정보**: 포켓몬에 대한 기술정보를 빠르게 조회할 수 있어 게임을 보다 편하게 공략할 수 있습니다.

## 기술 스택 (Tech Stack)

- **Frontend**: Next.js, React, TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Netlify
